# Portfolio

Professional portfolio website for Yatin Batra - Principal Software Engineer specializing in AI/GenAI products and cloud-native systems.

## Technology Stack

Pure HTML, CSS, and JavaScript. No build tools or frameworks required.

## Structure

**config.json** - All portfolio content (projects, writing, contact info)
**index.html** - Main page structure
**styles.css** - All styling
**app.js** - Content injection and interactions
**favicon.svg** - Site icon

## Updating Content

Edit config.json to update:
- Personal information and tagline
- Project descriptions and results
- Technical writing links
- Contact information
- Resume link visibility

Changes take effect immediately on page refresh.

## Deployment

Deploy to GitHub Pages:
1. Push files to GitHub repository
2. Enable GitHub Pages in repository Settings
3. Select main branch and root folder
4. Site will be live at your GitHub Pages URL

## Features

- Data-driven content from JSON
- Responsive design (mobile and desktop)
- Masked contact information (click to reveal)
- Technical monospace styling for badges
- Zero build process required

## Local Development

Open index.html directly in a browser. No server needed.

## Contact Masking

Email and phone are masked by default for security. Click to reveal full information. LinkedIn and GitHub links remain visible.
